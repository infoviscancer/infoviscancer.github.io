The Cancer Genoma Atlas


The Cancer Genome Atlas (TCGA) is a comprehensive and coordinated effort to accelerate the understanding of the molecular basis of cancer through the application of genome analysis technologies, including large-scale genome sequencing.


The overarching goal of TCGA is to improve our ability to diagnose, treat and prevent cancer.
The Cancer Genome Atlas (TCGA) Data Portal provides a platform for researchers to search, download, and analyze data sets generated by TCGA.


Dataset description


For the development of the project the TCGA dataset was considered relative to the Stomach adenocarcinoma (STAD), consisting of ~271 file type gene.quantification.txt (total size = 330 MB).
Each examined file refers to a specific cancer patient and has been generated by applying the technique of RNA-Sequencing, a methodology for the analysis of the transcriptome (set of RNA transcripts that characterize a given stage of development of a cell). Each file contains a list of the patient's genes (average ~24000) and for each of them the value of RPKM (Reads Per Kilobase per Million mapped reads), as well as other information.
The RPKM value represents a measure of gene expression obtained with normalization methods applied in the technique of RNA-Sequencing.

Data extraction


To extract the information of interest it was decided to create a Map-Reduce Job using the Apache Hadoop framework.

  • Map
  • Reduce
The Reduce phase takes as input the output of the mapper grouped by key, the key-value pairs where the key is formed by the Participant and the string value is a list of gene$RPKM associated with the key.
In this phase also define the form that will take the output of the job into the file system of Hadoop and into a csv file.
